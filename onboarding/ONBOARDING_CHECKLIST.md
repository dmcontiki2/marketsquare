# MarketSquare — Seller Onboarding Credential Checklist
**Manual onboarding reference · 15 April 2026**
*Use this during seller onboarding to know exactly which documents to request, upload, and verify for each category. Trust Score points are only awarded after manual review and approval.*

---

## How to use this checklist

1. Determine the seller's category (and sub-class for Services / Adventures)
2. Work through the checklist for that category
3. Request each document listed under **Request from seller**
4. Upload documents via the admin tool (Step 3 — listing form) or seller profile
5. Tick off **Verified** only after you have confirmed the document is genuine and (where applicable) the registration number is active in the relevant public register
6. Note any expiry dates in the **Expiry** column — calendar reminders must be set for re-verification

Scores update automatically in BEA when you approve credentials in the admin tool.

## Category: PROPERTY

### Regulatory note
PPRA registration is a legal requirement in South Africa to operate as a property practitioner. A seller who cannot provide a valid PPRA certificate should not be listed under Property.

**Public registers to check:**
- PPRA: [ppra.org.za](https://www.ppra.org.za) → Verify registration number is current and not suspended
- Professional bodies: IEASA, SAPOA — verify membership via their websites

| # | Document / Action | Request from seller | Verified | Expiry | Pts |
|---|---|---|---|---|---|
| P1 | PPRA / EAAB Registration Certificate | Certificate with registration number | ☐ | Annual renewal | 15 |
| P2a | NQF4 Real Estate qualification | Certificate issued by accredited provider | ☐ | None | 6 |
| P2b | NQF5 Real Estate qualification | Certificate | ☐ | None | +6 (12) |
| P2c | NQF6+ or professional designation | Certificate | ☐ | None | +8 (20) |
| P3 | Professional body membership (IEASA / SAPOA / NAR) | Membership card or letter | ☐ | Annual | 5 |

**Category max: 40 pts** | **Total possible (Universal + Category + Track Record): 100 pts**

### Scoring path for Property
| Score target | What to have |
|---|---|
| 40+ (Established) | ID verified + PPRA registration |
| 70+ (Trusted) | ID + PPRA + NQF5 + 6 months active + some intros |
| 90+ (Highly Trusted) | ID + PPRA + NQF6+ + pro body + 20+ intros + 5+ referrals |

## Category: TUTORS

### Regulatory note
SACE registration is required to teach in SA schools but is not a legal requirement for private tutors. It is a strong trust signal. Tutors without SACE can still reach Trusted status via qualifications and experience.

**Public registers to check:**
- SACE: [sace.org.za](https://www.sace.org.za) → Verify SACE number is active

| # | Document / Action | Request from seller | Verified | Expiry | Pts |
|---|---|---|---|---|---|
| T1 | SACE registration certificate | SACE certificate with registration number | ☐ | Annual | 8 |
| T2 | Highest academic qualification | Certificate (diploma, degree, honours, masters, doctorate) | ☐ | None | 6–14 |
| T3 | Subject specialisation evidence | Subject-specific certificate, transcript, or endorsement | ☐ | None | 5 |
| T4 | CV with teaching / tutoring experience | Structured CV — check dates and references are verifiable | ☐ | None | 5–11 |
| T5 | CV quality check | Review for structure, verifiable history, no unexplained gaps | ☐ | N/A | 2 |

**Qualification point guide:**
- Certificate or Diploma (NQF5–6): 6 pts
- Bachelor's Degree (NQF7): 10 pts *(replaces diploma pts)*
- Honours / Postgrad (NQF8+): 14 pts *(replaces bachelor's pts)*

**Category max: 40 pts**

### Scoring path for Tutors
| Score target | What to have |
|---|---|
| 40+ (Established) | ID verified + SACE + diploma/degree |
| 70+ (Trusted) | ID + SACE + degree + 5+ yrs experience + 6 months active |
| 90+ (Highly Trusted) | ID + SACE + postgrad + specialisation + 20+ intros + 5 referrals |

Professional body registration is the highest-weight signal. Always check the registration number is current — suspended or lapsed registrations score 0.

**Public registers to check:**
- ECSA: [ecsa.co.za](https://www.ecsa.co.za) → Engineering
- PIRB: [plumbing.org.za](https://www.plumbing.org.za) → Plumbing
- NHBRC: [nhbrc.org.za](https://www.nhbrc.org.za) → Home builders
- FSCA: [fsca.co.za](https://www.fsca.co.za) → Financial services
- SAICA: [saica.co.za](https://www.saica.co.za) → Accounting

| ST4a | Additional ticket 1 (First Aid, working at heights, confined space, rigging) | Certificate with expiry date | ☐ | Per ticket | +3 |
| ST4b | Additional ticket 2 | Certificate | ☐ | Per ticket | +3 |
| ST5 | CV — years of trade experience | Structured CV — verify employment dates | ☐ | None | 4–8 |
| ST6 | CV quality check | Structure, references, no unexplained gaps | ☐ | N/A | 2 |

**Experience points:** 3–7 years = 4 pts · 7+ years = 8 pts

## Complaint and Penalty Reference

When a complaint is received during or after onboarding:

| Complaint # | Points deducted | Action |
|---|---|---|
| 1st | −8 | Log in admin, notify seller |
| 2nd | −5 | Log, notify |
| 3rd | −3 | Manual review triggered |
| 4th | −2 | Manual review |
| 5th+ | −1 each | |
| **Maximum total** | **−22** | |

- **3+ complaints in 90 days** → flag account, pause listing pending David review
- **Safety concern** → immediate review, listing paused
- **Complaint older than 24 months** → half weight
- **Disputed complaint upheld** → remove complaint, restore pts
